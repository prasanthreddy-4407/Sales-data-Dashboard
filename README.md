# Sales Data Dashboard

## Overview
This project is a comprehensive Sales Data Dashboard built using **Power BI** to analyze and visualize sales performance for a fictional SuperStore. The dashboard provides actionable insights into sales trends, profitability, and customer behavior across different regions and product categories.

## Features
- **Key Performance Indicators (KPIs):** Total Sales, Total Profit, Quantity Sold, and more.
- **Visualizations:**
  - Sales by Category and Sub-category.
  - Regional Performance Analysis.
  - Customer Segment Breakdown.
  - Time-series Analysis of Sales Trends.
- **Interactivity:** Dynamic filtering and slicing by Region, Category, and Time Period.

## Technologies Used
- **Power BI Desktop:** For data modeling, visualization, and dashboard creation.
- **Data Source:** `SuperStore_Sales_Dataset.csv` / `SuperStore Sales DataSet.xlsx` (Excel/CSV).

## How to Use
1.  **Prerequisites:** Ensure you have [Power BI Desktop](https://powerbi.microsoft.com/desktop/) installed.
2.  **Open the Project:**
    - Download or clone this repository.
    - Open the `sales report.pbix` file in Power BI Desktop.
3.  **Interact:** Use the slicers and filters on the dashboard to explore the data.

## Data Source
The dashboard is powered by the included **SuperStore Sales Dataset**, which contains detailed transaction records including:
- Order Date & Ship Date
- Customer Segment & Region
- Product Category & Sub-Category
- Sales, Profit, Quantity, and Discount metrics
